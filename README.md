## DEV:

### How do I run this thing?

Run these three things in different terminals:
$ bundle exec rails server
$ yarn stylesheets
$ yarn javascripts

### How do I build this thing?

The Dockerfile should take care of that for you, just make sure you have docker
installed and run
$ make
